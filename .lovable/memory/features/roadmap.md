## Post-MVP Priority: Audio watermarking + Stripe Connect

## Layer 1 — Basic (low effort)
- Track ratings (post-purchase only)
- Public producer profiles (bio, social, genre)
- Wishlist / follow producers + notifications
- Offer system (private bids on unlisted tracks)

## Layer 2 — Monetization (medium effort)
- Commission model (15% platform fee via Stripe Connect)
- Producer subscription tiers (free 5 tracks / Pro unlimited + analytics)
- Bidding/auction for "1 of 1" tracks

## Layer 3 — Exclusivity & Trust (medium effort, high value)
- Audio watermarking (audiowmark invisible watermark)
- Leak reporting + watermark verification
- DJ verified badge (identity verification)

## Layer 4 — Community & Discovery (high effort)
- Genre charts (anonymous, monthly)
- Set feedback ("dropped, crowd 🔥") — private to producer
- Referral system (credits for new buyers)

## Layer 5 — Killer Features (long term)
- Sync licensing marketplace (YouTube, podcast, ads)
- Stems sales (remix rights, premium pricing)
- Label scouting mode (label role, direct contact)
